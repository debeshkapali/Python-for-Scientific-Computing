# Python for Scientific Computing - Learning Assignment

Simple examples demonstrating basic Python skills for scientific computing.

## Files

- `simple_analysis.py` - Basic Python script with functions, loops, and file writing
- `learning_notebook.ipynb` - Jupyter notebook with NumPy, Pandas, and Matplotlib examples

## How to Run

Install dependencies:
```bash
pip install numpy pandas matplotlib jupyter
```

Run the script:
```bash
python simple_analysis.py
```

Open the notebook:
```bash
jupyter notebook learning_notebook.ipynb
```

## What I Learned

- Basic Python: loops, conditionals, functions
- NumPy: arrays and simple calculations
- Pandas: creating DataFrames and filtering data
- Matplotlib: making simple plots
- Reading/writing files
